# Serenity

A Flutter meditation app — dual-theme night-sky / dawn aesthetic with bundled
Playfair/Inter/Outfit fonts, real audio playback pipeline, Hive-persisted
progress, subscription scaffolding, and iOS + Android builds.

## Quick start

```bash
# 1. Prerequisites: Flutter 3.41+, Xcode 16+, Android Studio (optional),
#    Python 3 (only to regenerate the app icon).
flutter doctor

# 2. Install deps + generate localisations.
flutter pub get
flutter gen-l10n

# 3. Run on iOS simulator.
open -a Simulator
flutter run -d <simulator-udid>   # or: flutter run (picks first device)

# 4. Run on Android emulator.
flutter emulators --launch Pixel_9_Pro_XL
flutter run -d emulator-5554
```

## Running the tests

```bash
# Unit + widget
flutter test

# Full end-to-end screenshot walk
flutter test integration_test/app_flow_test.dart -d <device>
```

The integration test writes a marker file at each screen and can be paired
with an external `xcrun simctl io … screenshot` loop — see the comments at
the top of `integration_test/app_flow_test.dart`.

## Regenerating the launcher icon

```bash
# Edit assets/branding/icon.png then regenerate all sizes:
dart run flutter_launcher_icons
```

The icon is a gold crescent moon + star field on a deep night-sky
gradient. Regenerate programmatically via `tools/generate_icon.py`.

## Design system

`design-system/MASTER.md` is the single source of truth for the visual
language. All UI code reads colors/typography/spacing through the provided
tokens rather than hardcoding values. A `pages/` subfolder is reserved for
per-page overrides (see `MASTER.md` §7).

## Localisation

- English (`lib/l10n/app_en.arb`) is the template.
- Chinese (`lib/l10n/app_zh.arb`) mirrors it.
- Run `flutter gen-l10n` after editing either file.
- The app switches locale via `Settings → Language`, falling back to the
  OS locale when set to `System default`.

## Theming

Two palettes ship — `AppPalette.night` and `AppPalette.dawn`. Access from
widget code with `context.palette.bgDeep` etc. Brand/semantic/glow
constants that read identically against any backdrop stay on `AppColors`.
Switch via `Settings → Theme`.
